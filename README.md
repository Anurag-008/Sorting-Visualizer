3. <a href="https://www.geeksforgeeks.org/insertion-sort/"> Insertion Sort </a>

5. <a href="https://www.geeksforgeeks.org/quick-sort/"> Quick Sort </a>
6. <a href="https://www.geeksforgeeks.org/selection-sort/"> Selection Sort </a>
